# 🎉 About

- 一个『企业级』管理系统（定制系统，非模板），使用Vue3+Vite5开发
- 顺德中专团委学生会媒体部定制、授权使用
- 使用『[TDesign](https://github.com/Tencent/Tdesign-vue-next)』组件库

# 🔨 Develop

```shell
# Install
npm install

# Test
npm run dev

# Build
npm run build

# LocalTest
npm run local:dev

# LocalBuild
npm run local:build
```
